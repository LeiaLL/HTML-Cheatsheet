HTML Cheatsheet

Basic HTML Structure
Tag Definition and Function
< !Document > Defines the document type
< HTML > Defines an HTML document
< head > Contains the metadata of the document
< title > Defines the title of the document, appears in the browser tab
< body > Defines the documents body
< !--...-- > Defines a comment
< p > Defines a paragraph element
< h1 > Defines a headline elements, ranging from h1 to h6 and decending in size
< hr > Defines a thematic change in the content with a horizontal break line
< br /> Inserts a single break line

Styles and Semantics
Tag Definition and Function
< style > Defines style information for a document, internal stylesheet
< div > Defines a section in a document, block element with no default styling
< span > Defines a section in a document, inline element with no default styling
< header > Defines a header for a document or section
< footer > Defines a footer for a document or section
< main > Specifies the main content of a document
< section > Defines a section or element in a document
< article > Represents a self-contained composition in a document
< aside > Used to mark additional information that can enhance another element but is not required for understanding such as quotes, comments & footnotes
< table > Inserts a table element
< tr > Inserts a table row into the table
< td > Inserts table data into the table row
colspan="" Specifies how many columns an element should span across
rowspan="" Specifies the number of rows a cell should span
< b > Bold text
< i > Defines a part of text in an alternated voice or mood
< em > Defines emphasized text
< strong > Defines important text
< selector style='declaration:value;' > Inline styling in HTML
< Link > Used to like external documents or file to the HTML document

Media
Tag Definition and Function
< figure > Specifies self-contained media content
< img > Used to embed an image in a document
< picture > Defines a container for multiple image resources
< video > Used to embed video content in a document
< audio > Used to embed sound content in a document
< a > Defines an anchor element used to create a hyperlink to web pages, files, email addresses & locations in the same page using the href="" attribute

Lists
Tag Definition and Function
< ul > Defines an unordered list
< ol > Defines an ordered list
< li > Defines a list item
< dl > Defines a description list
< dt > Defines a term/name in a description list
< dd > Defines a description in a description list

Attributes
Type Definition and Function
class="" Specifies one or more class names for an HTML element
id="" Defines an identifier (ID) which must be unique in the whole document
href="" Specifies the URL an element is linked to
src="" Specifies the URL of the media file to play

Forms and input
Tag Definition and Function
< form > Defines a section of a document containing normal content, markup, special elements called controls and labels
< input > Used to create interactive controls for web-based forms in order to accept data from the user
< textarea > Defines a multi-line text input control
< button > Represents a clickable button
< select > Used to create a drop-down list
< option > Used to define an item contained in a <select> element
< lable > Used to define a caption for an element in an HTML form
< map > Defines a client-side image map
< area > Defines an area inside an image map
< canvas > Defines a drawable region in the document to draw graphics via JavaScript
< figcaption > Defines a caption for a <figure> element
